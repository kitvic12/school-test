# School Test — Система тестирования учеников

## Обзор проекта

Flask-приложение для проведения тестирования учеников в компьютерном классе. Поддерживает два режима тестов:
- **sqrt** (квадраты) — вопросы по квадратам чисел от 10 до 30
- **powers** (степени) — вопросы по степеням двойки от 2⁰ до 2¹⁴

### Архитектура

| Файл | Назначение |
|------|-----------|
| `app.py` | Точка входа, инициализация Flask + SocketIO, определение маршрутов |
| `routes.py` | Все HTTP и WebSocket обработчики (регистрация учеников, отправка ответов, управление тестом) |
| `test_manager.py` | Управление состоянием теста (активен/завершён/финализирован) |
| `question.py` | Генерация вопросов и проверка ответов |
| `writer.py` | Запись/чтение данных учеников (`student.json`) и настроек (`settings.json`) |
| `data_manager.py` | Устаревший модуль расчёта оценок (не используется напрямую в `app.py`) |
| `auth.py` | Простая аутентификация учителя (хардкод-учётные данные) |
| `flask_logger.py` | Кастомное логирование запросов с разделением по типу клиента |
| `run.py` | Скрипт установки зависимостей и запуска сервера |

### Клиентские интерфейсы

| Папка | Назначение |
|-------|-----------|
| `static/student/` | Интерфейс ученика (index.html, script.js, style.css) |
| `static/teacher/` | Панель учителя (index.html, script.js, style.css) |

### Технологии

- **Backend:** Python 3, Flask, Flask-SocketIO, eventlet
- **Frontend:** Vanilla HTML/CSS/JS + Socket.IO client
- **Хранение данных:** JSON-файлы (`student.json`, `settings.json`)

## Запуск и использование

### Быстрый запуск

```bash
# Через скрипт автоматической установки
bash install_and_run.sh   # Linux/Mac
# или
python run.py             # напрямую
```

Скрипт создаёт виртуальное окружение `venv`, устанавливает зависимости из `requirements.txt` и запускает сервер.

### Ручной запуск

```bash
python -m venv venv
source venv/bin/activate  # Linux/Mac
# или venv\Scripts\activate (Windows)
pip install -r requirements.txt
python app.py
```

### Доступные адреса

| Роль | URL |
|------|-----|
| Ученик | `http://<local-ip>:5000` |
| Учитель | `http://127.0.0.1:5000/teacher` |

IP-адрес учителя настраивается в `settings.json` (`TeacherIP`). По умолчанию — `127.0.0.1`.

## Настройки

Файл `settings.json`:

```json
{
    "TeacherIP": "127.0.0.1",
    "Port": 5000,
    "TotalQuestions": 10,
    "Graduations5": 90,
    "Graduations4": 75,
    "Graduations3": 50
}
```

| Параметр | Описание |
|----------|---------|
| `TeacherIP` | IP-адрес, с которого разрешён доступ к панели учителя |
| `Port` | Порт сервера |
| `TotalQuestions` | Количество вопросов в тесте |
| `Graduations5/4/3` | Пороговые проценты для оценок 5, 4, 3 (ниже — оценка 2) |

## Учётные данные учителя

- **Логин:** `teacherEaVi`
- **Пароль:** `ChekingTest`

Заданы в `auth.py`.

## Рабочий процесс

1. Ученик открывает страницу, вводит ФИ и номер ПК
2. Учитель запускает тест (выбирает вариант: квадраты или степени)
3. Ученики получают вопросы по очереди, вводят ответы
4. После завершения теста учитель видит результаты и оценки всех учеников
5. Для нового теста — кнопка "Новый тест" (сбрасывает данные)

## Хранение данных

- `student.json` — данные учеников и состояние теста (создаётся автоматически)
- Данные сохраняются после каждого действия (регистрация, отправка ответа, завершение)
